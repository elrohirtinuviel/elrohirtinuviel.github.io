E-Comm

E-Comm is a microservice project that is written in GO. This project contains 7 different microservices.

These microservices are:
  • Database and Authentication Service: Fulfilling DB requests and authenticating users or sellers.
  • Bff & Frontend Service: Loading all frontend elements and making requests to other services.
  • Catalog Service: Loading all products in home the page and individual products in their own pages.
  • Search Service: Adding products to elasticsearch and retrieving products from it via search query.
  • Cart Service: Making requests to DB service to add products, get products from the cart table etc.
  • Payment Service: Responsible for starting a payment process using "iyzico api".
  • Order Service: Creating order objects that contains purchased products, total price, address, purchase time info and passing it to the database service.
